🛒 Atons Supermarket Sales Analysis
📌 Project Overview

This project analyzes sales performance, customer purchasing behavior, and profitability trends for Atons Supermarket using Excel, Power BI, and Tableau.

The goal was to transform raw sales data into interactive dashboards and actionable business insights.

🛠 Tools Used
Microsoft Excel
Power BI
Tableau
Data Cleaning
Data Visualization

📊 Key Insights
Total sales amounted to ₦186,029,949.74.
The North region contributed the highest sales share (27%).
Kano emerged as the top-performing city.
Technology was the most purchased category.
Phones recorded the highest purchase volume.
Tablets generated the highest profit margins despite lower purchase volume.
